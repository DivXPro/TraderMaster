
# 游戏概念及规则
1. 预测单元: 游戏中下注的最小单位，在K线图上符合指定的时间长度和指定价格范围这两个维度的单元。
2. 玩家：游戏中的参与者，通过下注K线图上显示的预测单元实现押注。
3. 押注: 每次押注都会有一个最小金额限制，不能低于这个金额。
3. 赔率: 每个预测单元都会有一个动态的赔率，根据当前市场价格和预测单元的价格时间比较使用BS模型计算得出。
4. 结算: 当K线到预测单元的时间点时，结算这些时间到达的预测单元上的玩家押注。如果市场价格在预测单元的价格范围内，玩家押注的金额会按照赔率计算出利润。如果市场价格不在预测单元的价格范围内，玩家押注的金额会被当作损失处理。
